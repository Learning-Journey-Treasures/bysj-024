# 基于SpringBoot的二手数码回收系统+原生微信小程序+论文参考示例

#### Description
技术栈+工具：SpringBoot + MyBatis + shiro +Jsp +IDEA2022 + 微信开发工具

系统角色：管理员、出售用户、回收用户

系统功能：管理员（用户管理、公告管理、回收分类管理、品牌管理、设备管理、订单管理、评价管理等）、出售用户（微信授权登录、通知公告、查看收购信息、出售、个人中心等）、回收用户（微信授权登录、发布设备、管理设备、订单信息、评价信息等）
